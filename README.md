# Menoh Go

Golang binding for [Menoh](https://github.com/pfnet-research/menoh)

## Requirements

- OS
  - Linux
  - Mac
  - ~~Windows~~ (TBD)
- Go 1.10+
- [Menoh](https://github.com/pfnet-research/menoh) 1.0.1+

## Install

After install Menoh, then

```bash
$ go get -u github.com/pfnet-research/menoh-go
```
